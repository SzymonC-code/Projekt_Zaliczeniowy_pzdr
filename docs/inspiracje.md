## Inspiracje
1. Aplikacja edukacyjna wspomagająca naukę (*quizy, powtórki, statystyki*)
2. Interaktywny katalog/muzeum online z opisami i multimediami
3. Prosty system zarządzania zadaniami (*to‑do, kanban*)
4. Portfolio projektów z możliwością dodawania i komentowania
5. Minimalistyczny dashboard analityczny dla małych projektów

### Krótka lista elementów projektu
- Interfejs użytkownika (*responsywny, dostępny*)
- Warstwa API / backend (*REST/GraphQL*)
- Baza danych (*relacyjna lub dokumentowa*)
- Autoryzacja i użytkownicy (*rejestracja, logowanie, role*)
- Obsługa plików i mediów (*upload, przeglądanie*)
- Testy jednostkowe i integracyjne
- Dokumentacja użytkownika i dewelopera
- CI/CD i prosty proces wdrożenia

### Wstępne dane wejściowe
- Zestaw przykładowych użytkowników (*admin, tester, user*)
- Przykładowe wpisy/rekordy (*CSV/JSON*) do zasilenia bazy
- Mocki API lub specyfikacja endpointów (*lista, szczegóły, tworzenie*)
- Zestaw grafik i ikon (*miniatura, obraz główny*)
- Plik konfiguracyjny projektu (*env przykładowy*)
- Mapowanie ról i uprawnień
- Scenariusze użycia / user stories (*priorytety*)
